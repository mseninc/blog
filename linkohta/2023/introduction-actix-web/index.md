## 想定環境

- Windows 11
- Rust 1.59 以降
- Actix Web 4

## Actix Web について

Actix Web は Rust 製の Web フレームワークです。

単体で HTTP サーバとして機能し、静的ファイルのやり取りや Web API を実装することができます。

また、動作が非常に軽量であることを特徴にしています。
