こんにちは。

この1年ほど密かに進めてきた、**社長LINEスタンプ**がいよいよ**審査中**となりました。
リジェクト（拒否）されなければ近日中にお披露目になります！

クリエイターズスタンプを購入されている方も多いと思いますが、**どうやって登録するの？**と思ったかたもおられると思いますので今回登録までに実施した流れを紹介したいと思います。

## スタンプ登録までの流れ

1. LINEアカウントを登録する
クリエイターズスタンプを登録しようとする方がLINEを使っていないということもないとは思いますが、LINEを使える状態になっていることが必須です。また、設定でメールアドレスを登録します。セキュリティ対策には十分注意してください。（参考画像はPC利用の設定画面です）
<a href="images/register-line-creators-stamp-1.png"><img src="images/register-line-creators-stamp-1.png" alt="" width="821" height="589" class="aligncenter size-full wp-image-4636" /></a>

1. クリエイターズ登録をする
[ここから](https://creator.line.me/ja/)クリエイターズページに進み、マイページからログインし、クリエイターズ登録をします。
<a href="images/register-line-creators-stamp-2.png"><img src="images/register-line-creators-stamp-2.png" alt="" width="1507" height="865" class="aligncenter size-full wp-image-4637" /></a>

1. スタンプを作成します。
弊社は、スタンプ作成専門業者に依頼しましたが、LINEスタンプを作る方法などの書籍は多数販売されておりますので自分にあった方法でスタンプを作成します。

1. 審査に出す
2のクリエイターズページにログインし、新規登録から順に進めます。以下に画像を使って流れを紹介します。

**新規登録を進める**
<a href="images/register-line-creators-stamp-3.png"><img src="images/register-line-creators-stamp-3.png" alt="" width="1507" height="865" class="aligncenter size-full wp-image-4641" /></a>
<a href="images/register-line-creators-stamp-4.png"><img src="images/register-line-creators-stamp-4.png" alt="" width="1507" height="865" class="aligncenter size-full wp-image-4642" /></a>

**登録画面を進める**
スタンプ詳細で、日本語の説明ができるように追加します。
<a href="images/register-line-creators-stamp-5.png"><img src="images/register-line-creators-stamp-5.png" alt="" width="1152" height="833" class="aligncenter size-full wp-image-4643" /></a>

英字と日本語の説明を追加します。英語は全角が含まないように注意してください。
<a href="images/register-line-creators-stamp-6.png"><img src="images/register-line-creators-stamp-6.png" alt="" width="1152" height="738" class="aligncenter size-full wp-image-4644" /></a>

**クリエイター情報とライセンス証明を登録する**
クリエイターの情報と、カテゴリを登録します。また、写真を利用している場合は著作権の観点で利用してよいことを証明する書類を添付する必要があるようです。弊社の登録では写真を利用しないため**スタンプに写真を利用していません**をチェックして進みます。
<a href="images/register-line-creators-stamp-7.png"><img src="images/register-line-creators-stamp-7.png" alt="" width="1163" height="1007" class="aligncenter size-full wp-image-4645" /></a>

**以下のようなテスト的な記述で保存**
<a href="images/register-line-creators-stamp-8.png"><img src="images/register-line-creators-stamp-8.png" alt="" width="1149" height="859" class="aligncenter size-full wp-image-4646" /></a>
<a href="images/register-line-creators-stamp-9.png"><img src="images/register-line-creators-stamp-9.png" alt="" width="1158" height="1017" class="aligncenter size-full wp-image-4647" /></a>

保存を押すと、Chromeでは以下のように**ページを離れていいか？**と聞かれますが**OK**で進みます。
<a href="images/register-line-creators-stamp-10.png"><img src="images/register-line-creators-stamp-10.png" alt="" width="1107" height="1038" class="aligncenter size-full wp-image-4648" /></a>

無事保存できました。
<a href="images/register-line-creators-stamp-11.png"><img src="images/register-line-creators-stamp-11.png" alt="" width="1145" height="963" class="aligncenter size-full wp-image-4649" /></a>

続いてスタンプを登録します。
<a href="images/register-line-creators-stamp-12.png"><img src="images/register-line-creators-stamp-12.png" alt="" width="1153" height="949" class="aligncenter size-full wp-image-4650" /></a>

登録するスタンプの枚数を変更し、ZIPでアップロードします。ファイルの中身は、`01.png～40.png` というファイル名で準備し、メイン画像とタブ画像のために、`main.png` と `tab.png` を登録します。計42枚の画像を**ZIP**で圧縮します。
<a href="images/register-line-creators-stamp-13.png"><img src="images/register-line-creators-stamp-13.png" alt="" width="1141" height="647" class="aligncenter size-full wp-image-4651" /></a>

販売価格は標準で120円に設定されていますが、変更する場合は以下の流れで変更します。
<a href="images/register-line-creators-stamp-14.png"><img src="images/register-line-creators-stamp-14.png" alt="" width="1151" height="779" class="aligncenter size-full wp-image-4652" /></a>

全ての登録が完了すると、右上のリクエストボタンが押せるようになるので、審査を申請します。以下の画像はテスト登録のため押せない状態になっています。
<a href="images/register-line-creators-stamp-15.png"><img src="images/register-line-creators-stamp-15.png" alt="" width="1161" height="972" class="aligncenter size-full wp-image-4653" /></a>

審査状態になるとアイテム管理から確認できます。
<a href="images/register-line-creators-stamp-16.png"><img src="images/register-line-creators-stamp-16.png" alt="" width="1456" height="651" class="aligncenter size-full wp-image-4654" /></a>

無事登録された際には、ブログで連絡します！
<a href="images/register-line-creators-stamp-17.png"><img src="images/register-line-creators-stamp-17.png" alt="" width="96" height="74" class="aligncenter size-full wp-image-4656" /></a>
